Star Wars Data Analysis — End-to-End Project
A complete data analysis pipeline built on a custom Star Wars universe database, created as part of a data analysis course. The project covers database design, Python-based analysis, and interactive business intelligence reporting.
Project Structure
This project was completed in four stages:
Stage 1 — Database Design (SQL): Designed and built a relational database from scratch in Microsoft SQL Server with four interconnected tables: Planets, Characters, Starships & Vehicles, and Weapons. Implemented primary keys, foreign keys, cascade rules, and data validation constraints.
Stage 2 — Data Analysis (Python / Jupyter Notebook): Queried the database and performed exploratory data analysis using Python. Generated charts and graphs to answer specific questions about the Star Wars universe data.
Stage 3 — Interactive Dashboard (Power BI): Built a four-page interactive dashboard covering planetary data, character profiles, weapons analysis, and starship statistics. Includes filters, KPI cards, and cross-table visualizations.
